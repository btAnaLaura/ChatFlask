Trabalho Final - Flask e Sockets

O que deve ser feito?
	
	Uma página web simples funcionando como uma interface web para comunicação com sockets TCP. Deve funcionar como um simples batepapo web (chat) como diversos que existem por ai.

Quais são as exigências?
	
	Os pontos bases são:

	1. Deve ser implementado em Python, obrigatoriamente;
	2. Deve conter sockets TCP, obrigatoriamente;
	3. A interface web deve ser desenvolvida utilizando Flask, obrigatoriamente.
	4. Deve conter uma página anterior onde o usuário escolherá o nickname antes de entrar no batepapo;
	5.As mensagens devem ser salvas no formato de log, em banco;
	6. Deve ser postado na plataforma Heroku ou Digital Ocean e deve estar online, portanto.
